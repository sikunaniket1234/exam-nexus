FAQ

Do students need to install any software?

No. ExamNexus is a 100% web-based SaaS. Students only need a browser (Chrome/Edge) and a webcam.

Can I use my own school's domain?

Yes. We provide subdomains (e.g., greenwood.examnexus.com) by default, but Enterprise plans allow custom CNAME mapping.

Is the data secure?

Absolutely. We use PostgreSQL with row-level security and automated daily backups. All data is encrypted in transit.

Does the AI generation work for Math?

Currently, the AI is optimized for text-based subjects (History, Science, English). Math support is in beta.
